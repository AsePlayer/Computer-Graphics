README for Project 7: Putting It All Together
By: Justin & Ryan

To Run:

1) Install required libraries:
- sudo apt-get install freeglut3-dev mesa-common-dev libglu1-mesa-dev libglew-dev libglfw3-dev libsoil-dev libglm-dev
- sudo apt-get install libfreetype-dev libfreetype6 libfreetype6-dev

2) Compile:
- g++ CheckeredTriangles.cpp -o triangles -lGL -lglut -lGLU
- g++ ColorCubeFlyby.cpp -o cubes -lGL -lglut -lGLU

3) Run:
- ./triangles
- ./cubes
